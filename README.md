# OS Page Replacement Project (React + HTML/CSS/JS)

This mini project simulates page replacement algorithms used in Operating Systems:

- FIFO
- LRU
- Optimal

## How to Run

1. Open `index.html` in your browser.
2. Enter:
   - algorithm
   - number of frames
   - page reference string
3. Click **Run Simulation**.

## Features

- Step-by-step frame table visualization
- Hit/Fault status per page request
- Total hits and faults
- Hit ratio and fault ratio

## Tech Used

- HTML
- CSS
- JavaScript
- React (via CDN)
